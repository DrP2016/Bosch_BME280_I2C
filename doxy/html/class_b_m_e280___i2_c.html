◆ osrs_mode_write()

void BME280_I2C::osrs_mode_write ( uint8_t  mode)

Write Oversampling Rate for T, P, H and Mode to BME280.

Parameters
[in]mode0b00-Sleep; 0b01-Forced; 0b11-Normal

Build the Byte, that contains Oversampling for T, P and Mode, then write it to 'BME280_REGISTER_CONTROL'

BME280_REGISTER_CONTROL [7:5] = 'osrs_t'

BME280_REGISTER_CONTROL [4:2] = 'osrs_p'

BME280_REGISTER_CONTROL [1:0] = 'mode'

Oversampling for Humidity is written directly to 'BME280_REGISTER_CONTROLHUMID'

BME280_REGISTER_CONTROLHUMID [2:0] = 'osrs_h'

void BME280_I2C::read_adc_burst ( void  )

Read adc_T, adc_P, adc_H in 'Burst Mode'.

Before reading, change BME280 Mode from 'Sleep' to 'Forced or 'Normal'

Reads adc_T, adc_P, adc_H by doing a very fast 'Burst Read' on 0xF7 to 0xFE

After reading, change BME280 Mode to 'Sleep'
